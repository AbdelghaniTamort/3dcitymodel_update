# Repo description

This repo contains the python scripts used in our semi-automatic pipeline for the update of buildings in a 3D city model using point-cloud based change detection. 
The used input data and the resulting outputs from our implementation are stored in the following drive : https://drive.google.com/drive/folders/1rngMvDyV8iJ5zsIP9jnML8CVBYcdaBb5?usp=drive_link
